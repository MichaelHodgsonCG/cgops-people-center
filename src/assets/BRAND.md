# Brand assets

Per the CG house style (see `CG_Dashboard` `src/assets/BRAND.md`): brand
assets are **imported as modules** from this folder; `publicDir` is disabled
in `vite.config.ts`, so nothing is served from a public directory.

- `cg-monogram.svg` — the CG monogram mark.
- `cg-wordmark.svg` — the Charcoal Group wordmark.

**Note:** the SVGs currently in this folder are neutral placeholders drawn to
the right proportions. Replace them with the official assets from
`CG_Dashboard/src/assets` (same filenames, no code changes needed).
